# ISO/IEC 27560 — Notice Receipt Extension

This folder holds the 0PN Lab profile extending **ISO/IEC TS 27560:2023**.

The canonical, openly-licensed source is the **Kantara ANCR** specification, published under Kantara WG IPR (RF-RAND):

**→ [Kantara ANCR — ISO-27560 TS Notice Receipt Extension](https://github.com/KantaraInitiative/ancr-wg/blob/d1a92ce689243708083e5723f02728b0cc00d810/27560%20ancr%20profile%20extension/ISO-27560%20TS%20Extension%20/27560%20TS%20Notice%20Receipt%20Extension.md)**

The Kantara ANCR specification extends TS 27560:2023 as an open, freely usable base — keeping the notice-receipt profile accessible independent of the closed ISO/IEC track.

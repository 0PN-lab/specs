## What this version is
UNR v1.1 is the working-draft enhancement set that expands the baseline profile toward:
- Privacy preference signal handling (for example GPC)
- Digital transparency risk assurance levels
- Rights automation patterns (where applicable)
- Token and exchange extensions (where applicable)
- AI transparency extensions (where applicable)

## How this fits the v1.2 consolidation
DCR v1.2 consolidates UNR v1 + v1.01 + v1.1 into a single “Digital Consent” packaging.
